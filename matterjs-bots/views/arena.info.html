Each arena comes with a Robot ID and Key pair, it can be used to run robot code directly on the server (server-side robot), instead of running it in browser (client-side robot).

Save robot code in this arena via the "upload" button in the Robot Control Panel. Reload the robot code using the "retrieve" button.

Once robot code has been uploaded, it can be loaded into any arena created using the Arena Builder as long as the Robot ID is known. Copy and paste the Robot ID into the Load Robots text area, or replace an existing line.

The Robot ID and Robot Key can also be used to directly upload/update the robot via the Robot Manager.

Sharing the Robot ID allows others to load the robot.

Sharing the Robot Key allows others to update the robot.
